Java 技术体系

Java SE 标准版

支持面向桌面应用

Java EE 企业版

为开发企业环境下的应用程序提供一套解决方案

Java ME 小型版

支持 Java 程序运行在移动终端上的平台

Java 特点

  1. Java 语言是面向对象的（OOP）
  2. Java 语言是健壮的，Java 的强类型机制、异常处理、垃圾的自动回收等是 Java 程序健壮性的重要保障
  3. Java 语言是跨平台性的
  4. Java 语言是解释性的

Java 运行机制

Java 程序运行在一个名为 JVM 的虚拟机中，每个平台有每个平台对应的虚拟机，所以一套 Java 程序可以运行在不同的平台上

JVM

  • Java 核心机制-Java 虚拟机（Java virtual mechine）
    1. JVM 是一个虚拟的计算机，具有指令集并使用不同的存储区域，负责执行指令、管理数据、内存、计算器，包含在 JDK 中
    2. 对于不同平台，有不同的虚拟机
    3. Java 虚拟机的机制屏蔽了底层运行平台的差别，实现了”一次编译，到处运行“

JDK

  1. JDK 的全程是：Java Development Kit
  2. JDK = JRE + Java 开发工具（java、javac、javadoc等）
  3. JDK 是提供给 Java 开发人员使用的，其中包含了 Java 的开发工具和 JRE

JRE

  1. JRE 的全称是：Java Runtime Environment
  2. JRE = JVM + Java 核心类库
  3. 如果只是想运行 Java 程序，只需要安装 JRE 即可

JVM、JDK、JRE 三者关系

  1. JDK = JRE + Java 开发工具
  2. JRE = JVM + Java 核心类库
  3. JDK = JVM + Java 核心类库 + Java 开发工具
